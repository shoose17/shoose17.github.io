    Markdown快速入门

    简介

    Markdown 是一种具有纯文本格式语法的轻量级标记语言，由John Gruber与Aaron Swartz于2004年创建。Markdown 的目标是实现「易读易写」。Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。编写的文档后缀为 .md, .markdown。

    下面我们来快速入门使用Markdown吧
